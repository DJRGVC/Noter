String Objects

Strings are a new object type that represent sequences of characters. These characters can be letters, digits, special characters, or spaces.

‚ú® Defining Strings

You tell Python you're working with a string by enclosing text in quotation marks (either single ' or double " quotes‚Äîjust be consistent!).

String Operations

String Concatenation and Multiplication Python 
# String concatenation - joining strings together
hi = "hello there"
name = "ana"

# Concatenate without space
greet = hi + name  
print(greet)  # Output: hello thereana

# Concatenate with space
greeting = hi + " " + name
print(greeting)  # Output: hello there ana

# String multiplication - repeat strings
silly = hi + (" " + name) * 3
print(silly)  # Output: hello there ana ana ana
‚ö†Ô∏è Important Note

The + operator doesn't automatically add spaces between strings. You must explicitly include spaces in your concatenation if you want them!

Key String Operations:

  • Concatenation Use + to join strings together
  • Multiplication Use * with a number to repeat a string
  • Parentheses Control the order of operations, just like in math

Input with input()

Getting User Input Python

üö® Critical Concept: Type Casting

The input() function always returns a string, even if the user types a number! If you want to perform mathematical operations, you must convert (cast) the input to int or float.

Example: If the user types "5", Python sees it as the string "5", not the number 5. Use int(input(...)) to convert it to a number.

Control Flow: Branching

Now that we can test conditions, we can make our programs make decisions! Branching allows your code to execute different instructions based on whether conditions are true or false.

The Simple if Statement

Basic if Statement Python 
if <condition>:
    <expression>
    <expression>
    ...

If the condition is True, execute the indented code block. If False, skip it and continue with the rest of the program.

The if-else Statement

if-else Statement Python 
if <condition>:
    <expression>
    <expression>
else:
    <expression>
    <expression>

Execute one block if the condition is True, otherwise execute the other block. You'll always execute exactly one of the two blocks.

The if-elif-else Statement

if-elif-else Statement Python 
if <condition>:
    <expression>
elif <condition>:
    <expression>
elif <condition>:
    <expression>
else:
    <expression>

Check multiple conditions in order. Execute the first block whose condition is True, then skip all the rest. The else is a catch-all for when none of the conditions are true.

üí° Key Point

With if-elif-else, you will always execute exactly one block, and it will be the first one whose condition is True. Once a condition is met, Python skips all remaining conditions.

Complete Example

Comparing Two Numbers Python 
x = float(input("Enter a number for x: "))
y = float(input("Enter a number for y: "))

if x == y:
    print("x and y are equal")
    if y != 0:
        print("therefore, x / y is", x/y)
elif x < y:
    print("x is smaller")
else:
    print("y is smaller")

print("thanks!")  # Always executes

Notice the nested if statement inside the first block‚Äîyou can nest conditionals as deeply as needed!

Indentation Matters!

üö® Critical: Python Uses Indentation for Structure

Unlike many programming languages that use braces {} or keywords to denote code blocks, Python uses indentation. This is not optional‚Äîit's how Python knows which code belongs to which block!

Indentation Rules:

  • Use 4 spaces for each level of indentation (standard convention)
  • All statements at the same indentation level are part of the same block
  • Indented code under an if, elif, else, or loop belongs to that structure
  • Code at the original indentation level is outside the conditional/loop
Indentation Example Python 
x = float(input("Enter a number for x: "))
y = float(input("Enter a number for y: "))

if x == y:
    print("x and y are equal")      # Part of outer if
    if y != 0:
        print("therefore, x / y is", x/y)  # Part of inner if
elif x < y:
    print("x is smaller")          # Part of elif
else:
    print("y is smaller")          # Part of else

print("thanks!")                    # NOT part of any conditional!

The beauty of Python's indentation is that it forces you to write readable code‚Äîwhat you see is what you get!

While Loops: Repeating Until a Condition Changes

Imagine you're playing a video game where you're stuck in a forest. You keep going right, and the same screen appears over and over. This is a perfect use case for a loop!

üéÆ The Lost Woods Problem

In the classic game The Legend of Zelda, there's a forest where if you keep going right, you stay stuck in the same place. The only way out is to go a different direction. How would we code this?

While Loop Syntax

While Loop Structure Python 
while <condition>:
    <expression>
    <expression>
    ...

How it works:

  1. Check if the condition is True
  2. If True, execute all expressions in the code block
  3. Go back to step 1 and check the condition again
  4. Repeat until the condition becomes False
  5. When False, skip the loop and continue with the rest of the program

Lost Woods Example

Lost Woods Game Python 
n = input("You're in the Lost Forest. Go left or right? ")

while n == "right":
    n = input("You're in the Lost Forest. Go left or right? ")

print("You got out of the Lost Forest!")

As long as the user keeps typing "right", they stay in the loop. Once they type anything else (like "left"), the condition becomes False and they escape!

While Loop with Counter

Counting with While Loop Python 
n = 0  # Initialize counter

while n < 5:
    print(n)
    n = n + 1  # Increment counter

# Output: 0, 1, 2, 3, 4
‚ö†Ô∏è Beware of Infinite Loops!

If you forget to increment your counter (or change the condition in some way), you'll create an infinite loop! Your program will run forever (or until you stop it with Ctrl+C or Cmd+C).

For Loops: Iterating Over Sequences

While loops are great when you don't know how many times you'll loop. But what if you want to do something a specific number of times? That's where for loops shine!

For Loop Syntax

For Loop Structure Python 
for variable in range(some_num):
    <expression>
    <expression>
    ...

How it works:

  • The loop variable takes on each value in the sequence, one at a time
  • For each value, execute all expressions in the code block
  • When the sequence is exhausted, exit the loop

Simple For Loop Example

Basic For Loop Python 
# Print numbers 0 through 4
for n in range(5):
    print(n)

# Output: 0, 1, 2, 3, 4

Much simpler than the while loop version! No need to initialize or increment a counter‚ÄîPython does it for you.

The range() Function

The range() function is incredibly flexible. It can take 1, 2, or 3 arguments:

range(stop)

range(5)

Generates: 0, 1, 2, 3, 4

Start: 0 (default)
Step: 1 (default)

range(start, stop)

range(2, 7)

Generates: 2, 3, 4, 5, 6

Step: 1 (default)

range(start, stop, step)

range(1, 10, 2)

Generates: 1, 3, 5, 7, 9

Every other number

üí° Important: Stop is Exclusive

The range() function always goes up to, but not including, the stop value. So range(5) gives you 0, 1, 2, 3, 4‚Äînot 5!

Range Examples

Different Range Configurations Python 
# Sum numbers from 7 to 9
mysum = 0
for i in range(7, 10):
    mysum += i
print(mysum)  # Output: 24 (7 + 8 + 9)

# Sum odd numbers from 5 to 9
mysum = 0
for i in range(5, 11, 2):
    mysum += i
print(mysum)  # Output: 21 (5 + 7 + 9)

The break Statement

Sometimes you want to exit a loop early, before the condition becomes false or before you've gone through all the values. That's where break comes in!

üõë How Break Works

When Python encounters a break statement:

  • Immediately exit the innermost loop it's in
  • Skip any remaining expressions in that loop's code block
  • Continue with the code after the loop
Break in Nested Loops Python 
while <condition_1>:
    while <condition_2>:
        <expression_a>
        break  # Exits inner while loop only
        <expression_b>  # Never executes!
    <expression_c>  # This DOES execute
